# 🚧 Graphics Settings

## Memória de Cache FIFO

Status: Planejado

Descrição:

Permite configurar o tamanho do buffer FIFO utilizado pelo sistema gráfico da Engine.

Opções:

- Auto
- Small (256 MB)
- Medium (512 MB)
- Large (1024 MB)

Objetivos:

- Melhorar a estabilidade.
- Reduzir travamentos (crash).
- Otimizar o uso da memória.
- Ajustar o desempenho conforme o dispositivo Android.
