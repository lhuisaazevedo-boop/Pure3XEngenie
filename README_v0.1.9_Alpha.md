## 🗺️ Roadmap

## 🚀 v0.2.0 Alpha — Foco Total Android

A partir desta versão, o desenvolvimento do Pure3XEngenie entrará em uma nova fase.

Durante as próximas semanas o projeto terá foco total na plataforma Android utilizando Android NDK r29.

Objetivos principais:

- Evolução completa do Android Framework
- Expansão do Android NDK Bridge
- Aprimoramento do JNI Bridge
- Melhor integração entre módulos Android
- Estrutura inicial do Frontend Android
- Melhor gerenciamento de Assets
- Organização do sistema de APK Loader
- Melhorias no sistema de inicialização
- Integração ARM64
- Expansão do sistema de configuração
- Melhor gerenciamento de memória
- Evolução do sistema de logs
- Preparação para Render Backend
- Melhorias na arquitetura modular
- Otimizações internas da Engine

## 🎯 Objetivo Atual

O foco principal do Pure3XEngenie neste momento é construir uma infraestrutura moderna, organizada e escalável para Android.

A prioridade absoluta das próximas versões será o desenvolvimento utilizando Android NDK r29, integração ARM64 e evolução dos componentes nativos da Engine.

A série 0.2.x será dedicada à consolidação da plataforma Android antes do avanço para etapas mais complexas da arquitetura de emulação.
